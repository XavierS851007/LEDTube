# LEDTube
LEDTube on Raspberry PICO W used as bedtime light

Website used to control the lights on the PICO based on a timing.
Red time means that kids should stay a litle longer in bed
Orange is that they can get out of bed but can't wake the parents
Green is time to wake up and get out of bed

timing can be changed between weekday and weekend
timing for the different colors can be changed
when green, the other leds will start to glow harder and harder every minute

at night a night light can be used that dimes sofly glowing orange/yellow
